## how to run

``` bash
# 安装依赖
npm install

# 端口改到了 8081
npm run dev
```

## 技术栈

-   使用 Vue-cli 脚手架初始化 Vue.js 项目
-   组件化、模块化的开发模式
-   模拟 json 后端数据，前后端分离开发
-   es6 + eslint
-   使用 scss 编写模块化的 CSS
-   使用 vue-router 解决路由问题
-   使用 vue-resource 与后端数据交互
-   使用第三方 JS 插件 better-scroll 进行交互
