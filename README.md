## how to run

``` bash
# 安装依赖
npm install

# 端口改到了 8081
npm run dev
```

## 技术栈

-   使用Vue-cli脚手架初始化Vue.js项目
-   组件化、模块化的开发模式
-   模拟json后端数据，前后端分离开发
-   es6 + eslint
-   使用scss编写模块化的CSS
-   使用vue-router解决路由问题
-   使用vue-resource与后端数据交互
-   使用第三方JS插件better-scroll进行交互
